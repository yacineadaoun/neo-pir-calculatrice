# Validité du protocole — NEO PI-R (Cabinet Pro)

Le manuel mentionne des **indicateurs de validité des réponses**. :contentReference[oaicite:9]{index=9}

## 1) Contrôles minimum (recommandés)
1. **Omissions** : nombre d’items vides.
2. **Style de réponse** : surutilisation d’une modalité (ex. trop de "N" ou répétition).
3. **Cohérence** : incohérences manifestes / réponses aléatoires (si indicateur disponible).

## 2) Décision
- **EN COURS** : protocole partiel, pas de rapport final.
- **INVALIDE** : seuils dépassés (omissions, style extrême, incohérence).
- **VALIDE** : complété + contrôles OK.

## 3) Paramétrage dans l’app
Les seuils doivent être :
- visibles
- modifiables
- tracés dans le rapport (audit clinique)
